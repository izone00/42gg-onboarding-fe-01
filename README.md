# 42gg 프론트엔드 온보딩 1단계

## 프로젝트 소개
- 기술 스택: Javascript, HTML, CSS
- todo 생성(Create), 조회(Read) 기능 구현하기 (새로고침 고려 X)
- todo 완료 상태 체크 기능 구현하기 (정렬은 선택사항) 
- todo 수정(Update), 삭제(Delete) 기능 구현하기 (새로고침 고려 X)
- 디자인 적용하기
- 무료로 배포하기
